This file contains the implementation of \mbox{\hyperlink{class_point_list}{Point\+List}} class. 

\mbox{\hyperlink{class_point_list}{Point\+List}} class, although similar to c++ vector tries to extend the default vector capabilities. The main reason for this class to exist is due to the requirement of sentinel nodes while computing the upper and lower tangents.
