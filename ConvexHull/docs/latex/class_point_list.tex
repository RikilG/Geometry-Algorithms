\doxysubsection{Detailed Description}
Structure to store a list of \mbox{\hyperlink{class_point}{Point}} class objects(points) 

This class gives functionality to store a list of \mbox{\hyperlink{class_point}{Point}} objects with additional support of having sentinel nodes which are auto determined by the class based on points present in its list.
