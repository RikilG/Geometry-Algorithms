\doxysubsection{Detailed Description}
Basic geometric \mbox{\hyperlink{class_face}{Face}} class to store a \mbox{\hyperlink{class_face}{Face}} structure. 

This \mbox{\hyperlink{class_face}{Face}} class has an id and a representative \mbox{\hyperlink{class_half_edge}{Half\+Edge}} which can be traversed to get the surrounding vertices and edges.
