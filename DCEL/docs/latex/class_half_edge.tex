\doxysubsection{Detailed Description}
Basic geometric \mbox{\hyperlink{class_face}{Face}} class to store a \mbox{\hyperlink{class_face}{Face}} structure. 

Each \mbox{\hyperlink{class_edge}{Edge}} will be split into 2 Half\+Edges and hence, these are directional. Each \mbox{\hyperlink{class_half_edge}{Half\+Edge}} object has a next pointer, prev pointer, twin pointer, tail pointer, head pointer and face pointer to give a complete information about all its incident geometric objects. This class is the main component of the \mbox{\hyperlink{class_d_c_e_l}{D\+C\+EL}} structure.
