\doxysubsection{Detailed Description}
Basic geometric \mbox{\hyperlink{class_vertex}{Vertex}} class to store coordinates of vertex and halfedge. 

This class stores the (x, y) coordinate of a vertex as taken from the user input. It also has a \mbox{\hyperlink{class_half_edge}{Half\+Edge}} object which stores a representative \mbox{\hyperlink{class_half_edge}{Half\+Edge}} for this vertex
