\doxysubsection{Detailed Description}
The main class for the \mbox{\hyperlink{class_d_c_e_l}{D\+C\+EL}} data structure. 

This is the main \mbox{\hyperlink{class_d_c_e_l}{D\+C\+EL}} data structure. All basic geometric structures are created dynamically and stored as pointers so that all structures can refer to each other without extra storage.
