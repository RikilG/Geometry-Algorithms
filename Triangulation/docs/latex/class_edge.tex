\doxysubsection{Detailed Description}
Basic geometric \mbox{\hyperlink{class_edge}{Edge}} class to store endpoints of edges. 

This class stores the 2 \mbox{\hyperlink{class_vertex}{Vertex}} type objects to represent an edge. Directionality is N\+OT assumed.
