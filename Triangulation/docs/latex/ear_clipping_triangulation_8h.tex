This function implements the basic ear clipping triangulation. 

The main approach behind this algorithm is that we try to find \char`\"{}ears\char`\"{} which are basically the vertices of the polygon whose interior angle is less than PI. We then triangulate these ears and remove them from the Polygon. we perform the same operation on the polygon until it is completely triangulated
